# OrderSummaryComponent
A clean, responsive, and fully accessible Order Summary Component built as part of the Frontend Mentor challenge.

---

## Overview
This project is a single-page Order Summary Component designed to display subscription details in a visually appealing card layout.
It is fully responsive and works across mobile and desktop screens. The component includes a hero illustration, plan information, and action buttons for proceeding to payment or canceling the order.

---

## Features
- Fully responsive layout for mobile and desktop
- Modern UI with smooth spacing and clean structure
- Reusable, semantic, and accessible HTML
- Custom styling using the **Red Hat Display** Google Font

---

## Built With
- **HTML5**  
- **CSS3** (Flexbox, responsive design)  
- **Google Fonts**
